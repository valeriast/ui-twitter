# Ui-Twitter

Twitter Clone using react.js, the project is not using any backend. It's only a ui clonse to learn react.js.
I'm using navigation with react-router-dom, icons from phosphor-react and some other components.

The purpose is to create a ui that it's able to temporarly save new tweets and display them in the content area. The same behaviour should happen with tweet answers.
